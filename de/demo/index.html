Live Demo

Automatisiertes E2E Monitoring

Was Sie nach einem Klick auf 'Start' sehen ist kein Video, sondern das Display eines Containers der live hoch gefahren wird. Im darin ablaufenden Test wird folgender Prozess von Sakuli durchgespielt und die Laufzeitmessungen werden im angezeigten Monitoring dargestellt:

  • Das zu testende System ist zum einen ein Web Ticket-Tool und zum anderen eine Windows Anwendung
  • Sakuli führt einen Prozess aus und verwendet dafür Web- und Bildbasierte Interaktion
  • Dabei werden verschiedene Rollen - ein End-User und eine Geschäftsperson simuliert (Sakuli loggt sich mit unterschiedlichen Zugangsdaten ein)
  • Die erzeugte Ticket-Nummer wird bei jedem Durchlauf in einem Excel-File gespeichert
  • Sakuli zeichnet ein Logo mit der Grafik-Anwendung Paint (dies dient natürlich nur dem Aufzeigen der Möglichkeiten von nativer Interaktion)
  • Der Eingang einer Bestätigungs-Email wird in Gmail geprüft
  • Schlussendlich werden die Laufzeiten der einzelnen Schritte an ein Monitoring System geschickt

Auf dem ersten Display können Sie den laufenden Test (via noVNC) verfolgen und auf dem zweiten Display werden die Monitoring Daten mit einer Grafana Oberfläche dargestellt. Hier können Sie zoomen, die Graphen verändern und unterschiedliche Sichten einstellen.
